# Ed-Donner-model-01---Chat-Model

## 🚀 How to Use This Workflow

Deploy this Agentic AI architecture to your local workspace in seconds by following these quick steps:

1. **Download & Unzip:** Click the green **Code** button at the top of this page, select **Download ZIP**, and extract the folder on your computer.
2. **Open n8n Workspace:** Open a new or existing canvas inside your n8n dashboard.
3. **Import the JSON File:** Click the menu icon (three dots) in the top-right corner, select **Import from File**, and choose the `.json` file from your unzipped folder.
4. **Configure & Run:** Connect your personal API keys to the LLM/Vector nodes and click **Execute Workflow** to start the pipeline.
